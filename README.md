# Still — personal attention guardian for Android

Still — это персональный страж внимания. Он бережно глушит цифровой шум прямо на устройстве: блокирует назойливые трекеры, нежелательный контент и бесконечные ленты, сохраняя интернет быстрым и спокойным.

## Что делает Still
- Фильтрует трекинг и рекламные домены локально, без отправки данных наружу.
- Приглушает нежелательный и откровенный контент мягким «Чистым режимом».
- Убирает короткие видео и бесконечные рекомендации с помощью «Фокус-капсулы».
- Даёт понятные инсайты о шуме: попытки отвлечь, попытки узнать лишнее, топ-сталкеры дня.

Still не про VPN — это инструмент тишины и внутренней гигиены интернета.

## Режимы внимания
- **Чистый режим** — скрывает нежелательные и откровенные материалы. Работает деликатно, без давления.
- **Фокус-капсула** — убирает короткие видео и ленты рекомендаций. Мессенджеры остаются. При включении автоматически активирует Чистый режим.

## Как пользоваться
1. Установите приложение на Android-устройство.
2. Откройте Still и включите главный тумблер фильтра.
3. При необходимости активируйте Чистый режим или Фокус-капсулу.
4. Смотрите «Панель внимания» и «Картина дня», чтобы видеть, кто пытается отвлечь или узнать о вас лишнее.

## Технически
- Android-приложение на Flutter с локальным фильтром DNS; трафик не уходит на внешние серверы.
- Сервис `DigitalDefenderVpnService` поднимает туннель и применяет блок-листы из `assets/blocklists/`.
- Windows: метод-канал логирует включение/выключение фильтра (заглушка).

## Структура
- `lib/` — Flutter UI и контроллер состояния.
- `android/` — нативный код Android.
- `windows/` — раннер Windows с обработкой метод-канала.
- `assets/blocklists/` — блок-листы, включая словари для Чистого режима и Фокус-капсулы.
