# StillNet — тихий интернет с защитой от рекламы и трекинга

StillNet — это лёгкое Android‑приложение, которое фильтрует DNS‑запросы и убирает лишний шум из интернета: рекламу, трекеры, навязчивую телеметрию. В основе — security engine: Zilant Prime, работающий на уровне DNS.

## Режимы защиты
- **Стандарт** — адаптивная фильтрация: реклама и трекинг урезаны, сайты работают стабильно.
- **Продвинутый** — усиленная фильтрация: блокируется больше рекламных и трекерных доменов, возможны редкие поломки сайтов.
- **ULTRA** — максимальная защита: блокируется максимум рекламы, трекинга и телеметрии, включая экспериментальные домены. Возможны поломки сайтов и приложений.

## Как пользоваться
1. Установите приложение на Android‑устройство.
2. Откройте StillNet и включите защиту на главном экране.
3. Выберите подходящий режим: Стандарт, Продвинутый или ULTRA.
4. Следите за статистикой запросов и заблокированных доменов на экране «Статистика и домены».

## Сборка
- Flutter: проект настроен для Android и Windows. Укажите путь к Flutter SDK в `android/local.properties` и `windows/flutter/generated_config.cmake` при сборке.
- Android: сервис `DigitalDefenderVpnService` поднимает локальный туннель и перехватывает DNS‑запросы, блокируя домены из `assets/blocklists/android_basic.txt`.
- Windows: метод‑канал логирует включение/выключение защиты (заглушка).

## Структура
- `lib/` — Flutter UI и контроллер состояния.
- `android/` — нативный код Android.
- `windows/` — раннер Windows с обработкой метод‑канала.
- `assets/blocklists/` — пример блоклиста.

Security engine: Zilant Prime — DNS‑фильтр, который стоит за StillNet.
