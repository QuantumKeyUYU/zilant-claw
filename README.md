# StillNet — calm DNS shield for Android

StillNet — это лёгкий DNS‑щит для Android, который снижает шум в интернете: режет часть рекламы, трекинг и навязчивую телеметрию прямо на устройстве. Без регистрации, без аккаунтов, без «облака».

## Что делает StillNet
- Блокирует часть рекламных и трекинговых доменов через локальный DNS‑фильтр.
- Помогает уменьшить фоновую телеметрию приложений.
- Работает на устройстве, не отправляет трафик на внешний сервер.

StillNet не обещает 100% блокировку всей рекламы и не заменяет мощные браузерные расширения вроде uBlock Origin. Это компактный щит первого уровня для всего устройства.

## Режимы защиты
- **Стандарт** — баланс защиты и стабильности: скрывает часть рекламы и трекеров, почти не ломает сайты.
- **Продвинутый** — блокирует больше рекламы и трекинга, отдельные элементы сайтов могут работать нестабильно.
- **ULTRA** — агрессивная фильтрация: реклама, трекинг, OEM‑телеметрия и экспериментальные домены. Возможны поломки сайтов и приложений.

## Как пользоваться
1. Установите приложение на Android‑устройство.
2. Откройте StillNet и включите защиту на главном экране.
3. Выберите подходящий режим: Стандарт, Продвинутый или ULTRA.
4. Следите за статистикой запросов и заблокированных доменов на экране «Статистика и домены».

## Технически
- Android‑приложение на Flutter с локальным VPN‑интерфейсом, перенаправляющим DNS‑запросы через фильтр на устройстве.
- Сервис `DigitalDefenderVpnService` поднимает туннель и блокирует домены из встроенных списков `assets/blocklists/`.
- Windows: метод‑канал логирует включение/выключение защиты (заглушка).

## Структура
- `lib/` — Flutter UI и контроллер состояния.
- `android/` — нативный код Android.
- `windows/` — раннер Windows с обработкой метод‑канала.
- `assets/blocklists/` — пример блоклиста.

Security engine: Zilant Prime — DNS‑фильтр, который стоит за StillNet.
